# MeidoPhotoStudio

A rewrite of MultipleMaids

## Building

### Required Libraries

These libraries are all found in COM3D2's Managed folder

Place these in a folder called `lib`

* `Assembly-CSharp.dll`
* `Assembly-CSharp-firstpass.dll`
* `Assembly-UnityScript-firstpass.dll`
* `Newtonsoft.json.dll`
* `UnityEngine.dll`
* `Ionic.Zlib.dll`
* `BepInEx.dll`
